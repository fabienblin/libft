# Libft

Librairie C - Projet de cursus

## Getting Started

Makefile will compile libft.a

### Prerequisites

gcc
make

### Compiling

Makefile will compile with -Wall -Wextra -Werror

```
make
```

It will compile the library into libft.a, used for all C following projects.
